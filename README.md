# Django Task Manager

Система управления задачами, разработанная на Django 4.0.8. Приложение позволяет создавать, отслеживать и управлять задачами с возможностью категоризации, установки приоритетов и дедлайнов.

## Функциональные возможности

- Создание и управление задачами
- Категоризация задач по статусам
- Установка приоритетов
- Отслеживание дедлайнов
- Система уведомлений
- Календарное представление задач

## Технологии

- Python 3.12
- Django 4.0.8
- SQLite
- HTML/CSS
- JavaScript

## Установка и запуск

### Предварительные требования

- Python 3.12+
- pip (пакетный менеджер Python)
- Git

### Установка

1. Клонируйте репозиторий:
```bash
git clone <https://github.com/vikachernenko/task_manager.git)>
cd <директория-проекта>
```

2. Установите и активируйте виртуальное окружение:
```bash
pip install pipenv
pipenv install
pipenv shell
```

3. Примените миграции:
```bash
python manage.py migrate
```

4. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

5. Соберите статические файлы:
```bash
python manage.py collectstatic
```

6. Запустите сервер разработки:
```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000

### Настройка переменных окружения

Создайте файл `.env` в корневой директории проекта:

```env
SECRET_KEY=your_secret_key
DEBUG=True
```

## Использование

### Создание задачи

1. Войдите в систему
2. Перейдите в раздел "Создать задачу"
3. Заполните форму:
   - Название задачи
   - Описание
   - Категория
   - Приоритет
   - Дедлайн
4. Нажмите "Сохранить"

### Управление задачами

- Просмотр списка задач
- Фильтрация по статусу и приоритету
- Редактирование существующих задач
- Отметка о выполнении
- Удаление задач

### Категории задач

- To Do
- In Progress
- Done

### Приоритеты

- High
- Medium
- Low
